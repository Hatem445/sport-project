# 🏋️ تطبيق متابعة التمارين الرياضية
## Fitness Exercise Tracking Application

---

## 📦 محتويات المشروع

### 1. **fitness_tracker.py** 🎯
- **الملف الأساسي للتطبيق**
- واجهة نصية (CLI) - قائمة تفاعلية
- يحتوي على فئات `Exercise` و `FitnessTracker`
- دعم كامل للعربية
- حفظ البيانات تلقائياً في JSON

**الميزات:**
- إضافة تمارين جديدة
- عرض جميع التمارين
- إحصائيات شاملة
- البحث والتصفية
- ملخص يومي
- حذف التمارين
- واجهة نصية سهلة الاستخدام

**الاستخدام:**
```bash
python fitness_tracker.py
```

---

### 2. **fitness_tracker_gui.py** 🖥️
- **نسخة الواجهة الرسومية**
- بناء على `fitness_tracker.py`
- واجهة رسومية حديثة باستخدام Tkinter
- علامات تبويب منفصلة لكل عملية
- استيراد وتصدير البيانات

**الميزات:**
- إضافة تمارين بشكل بصري
- جدول لعرض التمارين
- إحصائيات مرئية
- بحث متقدم
- استيراد/تصدير البيانات

**الاستخدام:**
```bash
python fitness_tracker_gui.py
```

## 🚀 طرق الاستخدام

### ✅ الطريقة 1: واجهة سطر الأوامر (الأفضل للمبتدئين)
```bash
python fitness_tracker.py
```
- قائمة تفاعلية بسيطة
- سهلة التتبع والفهم
- تعمل في أي مكان

### ✅ الطريقة 2: الواجهة الرسومية (الأفضل للاستخدام اليومي)
```bash
python fitness_tracker_gui.py
```
- واجهة بصرية جميلة
- جداول وعلامات تبويب
- أسهل للعمليات المعقدة

### ✅ الطريقة 3: البرمجة المباشرة
```python
from fitness_tracker import FitnessTracker

tracker = FitnessTracker()
tracker.add_exercise("جري", 30, 300, "جري")
tracker.get_statistics()
```

## 🎯 أنواع التمارين

- **مشي** - تمارين المشي والمشي السريع
- **جري** - الجري بسرعات مختلفة
- **سباحة** - السباحة والحركات المائية
- **رفع أثقال** - تمارين القوة
- **اليوجا** - تمارين المرونة والاسترخاء
- **دراجة** - الدراجة الهوائية والثابتة
- **تمارين منزلية** - البيت والحبل والضغط
- **أخرى** - أي نوع إضافي

## 💡 نصائح مفيدة

1. **ابدأ الآن** - أضف تمارينك الحالية
2. **راقب التقدم** - استخدم الإحصائيات
3. **حدد أهدافاً** - اجعلها واقعية وقابلة للقياس

## 🔧 استكشاف الأخطاء

### المشكلة: لا تظهر النوافذ في الواجهة الرسومية
**الحل:** تحقق من تثبيت Tkinter
```bash
python -m tkinter
```

### المشكلة: الأحرف العربية لا تظهر بشكل صحيح
**الحل:** استخدم Python 3.6+ مع ترميز UTF-8

### المشكلة: فقدان البيانات
**الحل:** تأكد من وجود `fitness_data.json` في المجلد
