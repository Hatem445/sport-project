# 🏋️ تطبيق متابعة التمارين الرياضية
## Fitness Exercise Tracking Application

سهل الاستخدام وبسيط التصميم - تطبيق شامل لمتابعة التمارين الرياضية والإحصائيات

---

## ⚙️ المميزات الرئيسية

### 1. **إضافة تمارين جديدة** ➕
- أضف اسم التمرين
- حدد مدة التمرين بالدقائق
- أدخل السعرات الحرارية المحروقة
- حدد نوع التمرين (مشي، جري، سباحة، إلخ)

### 2. **عرض التمارين** 📋
- اعرض قائمة كاملة بجميع التمارين المسجلة
- كل تمرين يعرض: اسمه، النوع، المدة، السعرات، والتاريخ

### 3. **الإحصائيات الشاملة** 📊
- إجمالي عدد التمارين
- إجمالي الوقت المستغرق (بالساعات والدقائق)
- إجمالي السعرات المحروقة
- متوسط مدة التمرين
- إحصائيات لكل نوع تمرين على حدة

### 4. **البحث والتصفية** 🔍
- البحث حسب نوع التمرين
- البحث حسب التاريخ
- الملخص اليومي لأي تاريخ

### 5. **إدارة البيانات** 💾
- حفظ تلقائي للبيانات في ملف JSON
- تحميل البيانات السابقة عند فتح التطبيق
- حذف التمارين القديمة

---

## 🚀 كيفية الاستخدام

### التثبيت
```bash
# لا يتطلب مكتبات خارجية - استخدم Python 3.6+
python fitness_tracker.py
```

### أمثلة على الاستخدام

#### مثال 1: إضافة تمرين
```
اختر: 1
اسم التمرين: الجري الصباحي
مدة التمرين بالدقائق: 30
السعرات المحروقة: 300
نوع التمرين: جري
```

#### مثال 2: عرض الإحصائيات
```
اختر: 3
# سيعرض إجمالي التمارين والوقت والسعرات
```

#### مثال 3: البحث حسب النوع
```
اختر: 4
أدخل نوع التمرين: جري
```

---

## 📁 البيانات المحفوظة

يتم حفظ جميع البيانات تلقائياً في ملف `fitness_data.json` في نفس مجلد التطبيق.

مثال من الملف:
```json
[
  {
    "name": "الجري الصباحي",
    "duration": 30,
    "calories": 300,
    "exercise_type": "جري",
    "date": "2024-03-12 08:30:45"
  }
]
```

---

## 🎯 أنواع التمارين المقترحة

- **مشي** - تمارين المشي والمشي السريع
- **جري** - الجري بسرعات مختلفة
- **سباحة** - السباحة والحركات المائية
- **رفع أثقال** - تمارين القوة والأوزان
- **اليوجا** - تمارين الاسترخاء والمرونة
- **دراجة** - الدراجة الهوائية والثابتة
- **تمارين منزلية** - البيت، الحبل، الضغط، إلخ
- **أخرى** - أي نوع آخر

---

## 💡 نصائح مفيدة

1. **تقدير السعرات**: استخدم متوسط تقريبي بناءً على نوع التمرين والشدة
2. **التواريخ**: يتم تسجيل التاريخ والوقت تلقائياً
3. **النسخ الاحتياطية**: احتفظ بنسخة من `fitness_data.json` للأمان
4. **الانتظام**: كلما أضفت بيانات أكثر، كانت الإحصائيات أدق
