# 🏋️ FitLife AI - موقع اللياقة الذكي

## 📚 نظرة عامة

**FitLife AI** هو موقع ويب متقدم لمتابعة اللياقة والصحة مع تفاعل ذكاء اصطناعي متطور. يوفر:

- 🤖 **ذكاء اصطناعي متقدم** لتحليل البيانات والتوصيات الشخصية
- 💪 **متابعة شاملة** للتمارين الرياضية
- ❤️ **مراقبة صحية مستمرة** مع تحليل المؤشرات الحيوية
- 🏥 **استشارات طبية معتمدة** من السجلات الطبية الدولية
- 📱 **إشعارات ذكية** عبر SMS لتذكير بالتمارين والحالة الصحية
- 🎨 **واجهة داكنة مريحة** مصممة بعناية
- 📊 **رسوم بيانية تفصيلية** لتتبع التقدم

---

## ✨ المميزات الرئيسية

### 1. 📋 متابعة التمارين
- تسجيل التمارين بسهولة (اسم، نوع، مدة، شدة، سعرات)
- تاريخ وملاحظات لكل تمرين
- تحليل تلقائي لأداء التمارين
- جدولة التمارين مع إشعارات تذكيرية

### 2. ❤️ المراقبة الصحية
- تسجيل البيانات الحيوية (معدل النبض، الوزن، ضغط الدم، نسبة الدهون)
- مراقبة النوم والإجهاد والطاقة
- كشف اتجاهات الصحة تلقائياً
- تنبيهات فورية عند وجود شذوذ

### 3. 🤖 التوصيات الذكية

#### تطوير العضلات
- تحليل مجموعات العضلات المستهدفة
- توصيات ذكية لتطوير كل مجموعة
- بناء برامج تدريب مخصصة

#### صحة الأعصاب والتنسيق
- تمارين لتحسين التوازن
- تحسين المرونة والتنسيق
- يوجا وتمارين استرخاء

#### تشخيص الإصابات
- كشف الإصابات المحتملة
- توصيات العلاج الفوري
- خطط إعادة التأهيل

#### استشارات طبية معتمدة
- مراجع من:
  - **WHO** (منظمة الصحة العالمية)
  - **American Heart Association**
  - **American Academy of Sports Medicine**
  - **Mayo Clinic Guidelines**
  - **Orthopaedic Research Society**

### 4. 📱 نظام الإشعارات

#### تنبيهات صحية
```
⚠️ تنبيه: معدل ضربات القلب: 140 نبضة/دقيقة
الإجراء: توقف عن التمرين واستشر طبيباً
```

#### تذكيرات التمارين
```
🏋️ تذكير: لديك تمرين "الجري الصباحي"
⏰ الموعد: غداً الساعة 06:00 صباحاً
```

#### ملخصات يومية
```
📊 ملخص حالتك الصحية
- التمارين: 2 تمرين
- السعرات: 600 سعرة
- معدل النبض: 75 نبضة
- النوم: 7.5 ساعات
```

### 5. 📊 التحليلات والرسوم البيانية
- رسوم بيانية متقدمة للمؤشرات الصحية
- مقارنة البيانات عبر الفترات الزمنية
- تتبع التقدم نحو الأهداف
- تقارير شاملة وسهلة الفهم
